- 오늘 날짜 표시
- 아래 + 버튼을 누르면 입력 창이 뜬다.
- 할 일을 추가, 체크, 삭제 할수 있다.
- 모두보기, 완료된 일정, 아직 완료하지 않은 일정을 나누어 볼 수 있다.
- 남은 할일의 갯수를 확인할 수 있다.

! 개선사항
- 인풋 입력 후 엔터 누르면 두개씩 입력됨
- 체크 되었을 때 삭제 안됨
- 체크된것 할일 갯수 카운팅 안됨
